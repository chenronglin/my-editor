# 小说编辑器使用指南

#### 测试网站访问地址：

https://pub-3de432deba82409eb14e5e9990eccb2b.r2.dev/editor/index.html

## 1. 角色身份（编辑 vs 作者）

编辑器顶部最左侧是**身份切换区**。
- **编辑A**：负责对小说进行审阅、修改和添加批注。
- **作者B**：小说的创作者，负责编写初稿、查看编辑的修改意见并针对批注进行解释或修改。

> [!NOTE]
> 切换身份即可模拟不同角色在各自电脑上的操作视角和操作权限。

---

## 2. 工作模式 与 阅读模式（显示模式）

顶部右侧是**显示模式切换区**，用来控制正文和修订内容的显示方式：

| 模式名称     | 解释                                                         | 适合的使用场景                                     |
| ------------ | ------------------------------------------------------------ | -------------------------------------------------- |
| **工作模式** | 能够清晰地看到谁删除了什么、谁新增了什么。删除的内容带有红色删除线，新增的内容带有绿色下划线。 | 适合编辑进行修改，或者作者查看编辑修改了哪些地方。 |
| **阅读模式** | 自动隐藏所有的删除线内容，新增的内容也融入正常文本中，看起来就像是已经定稿发表的书籍一样。 | 适合编辑和作者通读小说，检查排版和语句流畅度。     |

---

## 3. 开启修订 与 关闭修订（修订模式）

当您以**“编辑A”**身份处于**“工作模式”**下时，工具栏会多出一个**【开启修订】**的按钮。

- **开启修订**：
  - 编辑点击此按钮后，正文会进入“修订模式”，此时**正文将被锁定，作者暂时无法修改正文**。
  - 编辑在正文中所做的所有改动（增加字句、删除字句）都会被记录下来，并以红绿标记显示。
  - **通俗理解**：编辑开启了“修改草稿”时间，通知作者“先别动正文，等我改完”。
- **关闭修订**：
  - 编辑修改完毕后，点击**【关闭修订】**。
  - **正文将被解锁，作者可以重新编辑正文**。
  - **通俗理解**：编辑改完了，把主动权交还给作者，作者可以根据编辑的意见继续写书。

---

## 4. 批注与评论区操作（划词与回复）

这是编辑与作者沟通的另一个方式。**无论是编辑还是作者，无论处于工作模式还是阅读模式**，都可以使用此功能。

### 新增批注（划词批注）
1. 用鼠标在正文中**选中（高亮蓝色）**一段需要讨论的文字。
2. 此时文本上方会弹出一个悬浮小工具栏，点击最右侧的**“添加批注（气泡加号图标）”**按钮。
3. 页面会弹出批注输入对话框，输入您的意见（例如：“这里描写太平淡了，建议增加心理描写。”），然后点击**【批注】**提交。
4. 被批注的文字在正文中会被标记为**黄色高亮**。

### 回复批注
1. 点击正文中的**黄色高亮文字**，或者点击顶部/右侧的**“显示批注（气泡图标）”**按钮打开右侧的**【批注区】**。
2. 在右侧对应的批注卡片下方，直接在输入框中输入您的回复（例如作者回复：“好的，已按建议修改。”），然后点击发送图标。
3. 编辑和作者可以在此处进行多次对话，形成讨论串。

### 删除批注
- 如果讨论结束或批注已无用，可以点击批注卡片上的**【垃圾桶（删除）】**按钮。您可以选择删除整条“批注串”或仅删除某一条特定回复。
